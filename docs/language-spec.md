# Language Specification (Draft)

## Program Structure

An SPL program consists of:

- Declarations
- Statements
- Expressions
- Control constructs

## Core Constructs

- Variable declarations
- Function definitions
- Conditional expressions
- Iterative constructs
- Pattern-based execution (planned)
